< img >
  • purpose: embeds an image into the document
  • example: <img src=../fluffy_cat.jpp" height="20%" alt="sweet fluffy ca">
  • attributes:
Attribute Value Meaning
src URL source file
height number,percentage The intrinsic height of the image,in pixels. Must be an interger without unit.
width number, percentage The intrinsic width of the image in pixels. Must be an interger without a unit.
alt text Defines an alternative text desrciption of the image.
< a > , </a>
  • purpose: anchor element, creates a hyperlink to web pages, files, email address, locations int the same page, or anything else a RL can address.
  • example: <a href="https://en.wikipedia.org/wiki/cat" >Wiki Cats.</a>
  • attributes:
Attribute Value Meaning
href URL The URL that hyperlink points to.
target name_of_frame Where to display the linked URL, as the name for a browsing context (a tab, window, or <ifrmae > ).
_blank usually a new tab, but users can configure browsers to open a new window instead.
_parent the parent browsing context of the current one. If no parent, behaves as _self.
_top the top most browsing context (the "highest" context thata TMs an ancestor of the current one). If no ancestors, behaves as _self.
_self the current browsing context. (Default)
